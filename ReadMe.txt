PicResize 1.0.2
A simple image resizer.
Instructions:
- Place a shortcut to the program on your desktop or in a folder
- Drag and drop an image on the shortcut
- Input height or width and hit Enter or click on “Set”
- Click on “Resize” to generate a resized copy of the image
- The image file name will contain the new dimensions
- You will find the new image file in the same folder as the original image file
